# Computer Network MCQs

This project is a simple HTML-based web page that provides multiple choice questions (MCQs) for Computer Networks.

## 📚 Features
- Easy to use interface
- Practice-based MCQs
- Useful for students and beginners
- Runs directly in browser (no installation required)

## 🚀 How to Use
1. Download or clone this repository
2. Open the `index.html` file in your browser
3. Start practicing MCQs

## 🎯 Purpose
This project is created for students who want to practice Computer Network concepts in a simple and interactive way.

## 👨‍🏫 Target Users
- ITI Students,computer instructor 2026
- Beginners in Computer Networks
- Competitive exam learners

## 🔧 Technologies Used
It includes the CSS, JavaScript, and HTML structure. It uses Tailwind CSS for styling and Canvas Confetti for the animations,
